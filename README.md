### 1\. Clone this Repository

```bash
git clone https://github.com/sunset-ventures/
```

### 2\. Navigate to the directory

```
cd sunset-ventures
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
