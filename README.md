### 1\. Clone this Repository

```bash
git clone https://github.com/0xaloysius/0xaloysius.git
```

### 2\. Navigate to the directory

```
cd 0xaloysius
```

### 3\. Install dependencies

```
yarn install
# or
npm install
```

### 5\. Run Dev Server

```
yarn dev
# or
npm run dev
```
